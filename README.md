## 遊び方
(リリースのURL)からワールドをダウンロードして、.minecraft/saves内に置くと遊べます。  
マルチプレイの場合はリソースパックも個別にDLしてご利用ください。  
サーバーの設定でコマンドブロックは利用可能にしておいてください。
